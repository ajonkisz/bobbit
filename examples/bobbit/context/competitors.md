# Competitor Landscape

| Tool | What It Does | Our Advantage |
|------|-------------|---------------|
| [Gas Town](https://github.com/steveyegge/gastown) | Multi-agent workspace manager | We focus on product coherence, not just orchestration |
| [Overcode](https://github.com/mkb23/overcode) | AI agent monitor/supervisor | We're product-owner-first, not developer-first |
| CLAUDE.md (manual) | Static context files hand-edited | We make these living documents with a UI, versioning, and context injection rules |
| Linear / Jira | Project management | We manage agent workforces, not human sprints |
| Cursor / Windsurf | AI-assisted coding | We sit above the coding layer — managing what to build, not how |

## Key Insight

Code generation is commoditized. The bottleneck is **coherence** — keeping multiple
agents aligned with product vision, design decisions, and each other. That's what
Bobbit solves.
