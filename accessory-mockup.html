Existing Accessories for Reference

How counter-hue-rotate works: The bobbit body uses filter: hue-rotate(Ndeg) to create session identity colors. Accessories inside the rotated container apply filter: hue-rotate(-Ndeg) to cancel out the rotation, keeping gold stars gold and purple hats purple regardless of the session's color. Each card shows the hue rotation value and the resulting bobbit color.
